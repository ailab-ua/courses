UA/Eller MIS Analytics Curriculum (Recommendations highlighted in boldface)

Analytics Working Group members: H. Chen, G. Leroy, B. Zhang, Y. Ge, S. Youn. Date: January 23, 2020

  BS

  1. MIS 301 Data Structures (Junming); need to focus on Python, syntax, ADT, searching/sorting, library, data, statistics, visualization (MIT Guttag book)
  2. MIS 331 DBMS (Bin); relations, SQL
  3. * (asterisk indicates prerequisite) MIS 464 Data Analytics (H Chen, Yong); web/data/text mining, DM algorithms, deep learning (CNN, RNN); possible for two semesters MIS 464A & MIS 464B (Fall, Spring)
  4. MIS 373 Operations Management
  5. * MIS 4XX/5XX, Optimization/Dynamic Programming (Youn); possible new class by Youn for OM and MIS BS and MS students
  6. MIS 461/561, Data Visualization (Lusi); Tableau, Python/R viz, Tufte viz; need to go beyond Tableau, possible for BS and MS students (with less technical background)
